Botones

Variantes de Botones

HTML
<button class="boton boton_primario">Primario</button>
<button class="boton boton_secundario">Secundario</button>
<button class="boton boton_outline">Outline</button>
<button class="boton boton_primario" disabled>Deshabilitado</button>
CSS
.boton {
  display: inline-flex;
  align-items: center;
  gap: 8px;
  padding: 12px 18px;
  border-radius: 999px;
  border: 1px solid var(--outline);
  text-decoration: none;
  color: var(--text);
  transition: .2s transform, .2s background, .2s border, .2s color, .2s box-shadow;
  cursor: pointer;
  font-weight: 700;
  position: relative;
  overflow: hidden;
}

.boton:hover {
  transform: translateY(-2px);
}

.boton:focus-visible {
  outline: 2px solid var(--primary);
  outline-offset: 2px;
  box-shadow: 0 0 0 4px rgba(212,175,55,.2);
}

.boton:disabled {
  opacity: .5;
  cursor: not-allowed;
}

.boton_primario {
  background: var(--primary);
  color: #1a1a1a;
  border-color: transparent;
}

.boton_primario:hover {
  background: var(--primary-700);
}

.boton_outline {
  background: transparent;
}

.boton_secundario {
  background: #1e1f24;
}
